Innovation

 

Connect

A B2B web portal for your customers to order online and track the order status.​

Engage

Our client was using SAP for business already, so we created a technical user account in SAP RFC communication & used SAP RFC​ system to sync data between SAP and our​ web portal.​

Transform

  • Products, discounts, packaging details, delivery details are obtained from the SAP system and presented in our web portal.
  • The customer can order and track the order status in the portal.
  • The orders from customers and all the transactions from the web portal are also captured in the SAP system without disturbing the existing SAP workflow.​
